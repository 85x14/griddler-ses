# Griddler::Ses

This is a [Griddler](https://github.com/thoughtbot/griddler) adapter that allows you to parse email replies when used with Amazon SES.


## Installation

Add these lines to your application's Gemfile:

```ruby
gem 'griddler'
gem 'griddler-ses'
```

And then execute:

    $ bundle
